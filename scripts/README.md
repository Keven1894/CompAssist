# Scripts directory

This directory contains platform-specific scripts that can be executed from the Docker container.

## Windows Scripts

- `windows/check-updates.ps1` - Check for Windows updates
- `windows/install-software.ps1` - Install software packages
- `windows/fix-issues.ps1` - Fix common Windows issues

## Linux Scripts

- `linux/check-updates.sh` - Check for Linux updates
- `linux/install-software.sh` - Install software packages
- `linux/fix-issues.sh` - Fix common Linux issues

## Usage

These scripts are automatically called by the assistant when running in Docker mode.
