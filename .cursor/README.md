# .cursor Directory

This directory contains configuration and rules for the Cursor AI agent working on this project.

## Files

- **`rules.md`** - Comprehensive rules and guidelines for the CompAssist agent
- **`agent-config.md`** - Agent configuration and quick reference

## Agent Identity

**Name**: CompAssist (Computer Assistant Agent)

**Purpose**: Assist with local computer environment management, setup, and maintenance tasks.

## Usage

These rules guide the AI agent when working on this project. They ensure:
- Consistent code style and practices
- Safe operations with proper confirmations
- Clear documentation and communication
- Platform-aware implementations

## Modifying Rules

If you need to update agent behavior or add new guidelines:
1. Edit `rules.md` for detailed rules
2. Update `agent-config.md` for quick reference changes
3. Commit changes to version control
