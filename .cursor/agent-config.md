# CompAssist Agent Configuration

## Agent Information

- **Name**: CompAssist (Computer Assistant Agent)
- **Version**: 1.0
- **Project**: Local Computer Assistant
- **Created**: 2024

## Project Context

This agent is designed to help manage and standardize computer environments across Windows and Linux systems. It provides automated tools for:

- Environment checking and verification
- Software installation and setup
- System updates and maintenance
- Problem detection and resolution
- Agent-to-agent communication via REST API and MCP
